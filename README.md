# Cozy TODO
Cozy-TODO is a sleek and highly customizable To-Do list app that lets you make productivity your own.
Change the background, and your favorite music and even rename the app perfect for keeping multiple tabs open with diffrent task list and vibes!

# Features
**Customizable Background**
**Build in Music Player**
**Multiple TO-Do Tabs**
**Persistent Data**
**Fast Lightweight**

# Tech Stack
**Frontend:** HTML, CSS, JavaScript
**Backend:** Node.js
**Package Management:** NPM
