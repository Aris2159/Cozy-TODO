# Sound Files Directory

This directory contains the default sound files for the Cozy Todo application. Please add your own sound files with the following names:

- `nature.mp3`: Calming nature sounds (birds, forest, etc.)
- `rain.mp3`: Gentle rain sounds
- `cafe.mp3`: Cafe ambience
- `ocean.mp3`: Ocean waves

## File Requirements

- Format: MP3 or WAV
- Duration: Recommended 1-5 minutes (will be looped)
- Quality: 128-320 kbps
- Size: Keep under 5MB per file for optimal performance

## Adding Custom Sounds

1. Download or create your sound files
2. Rename them to match the names above
3. Place them in this directory
4. Restart the application if it's running

Note: The application will use these sounds when "Default Sounds" is selected in the audio settings.
